# SheetToolBox デスクトップ版プロトタイプが完成しました

メンバーの皆さんにご報告です。

これまで個別に開発してきたスプレッドシート関連ツール3つを、**1つのデスクトップアプリ「SheetToolBox」** としてまとめたプロトタイプが完成しました。

## SheetToolBox に含まれるツール

| ツール | 機能 |
|--------|------|
| **SheetPic** | 必要なシートだけを抽出して新しいファイルを生成。書式・罫線・結合セルも完全保持 |
| **MacroRemover** | ファイルからマクロを一括除去。.xlsm → .xlsx の安全な変換 |
| **SheetMerge** | 複数ファイルのシートを1つのファイルに結合 |

## なぜデスクトップアプリにしたのか

Web版は手軽に使える反面、**ファイルをサーバーにアップロードする必要**がありました。

業務で扱うスプレッドシートには個人情報や社外秘のデータが含まれることも多く、「外部サーバーに送りたくない」という声を実際にいただいていました。

SheetToolBox は **Tauri + Rust** で構築しており、**ファイルが一切外に出ません**。すべての処理がローカルで完結します。

## 技術的なポイント

- **フレームワーク**: Tauri v2 + Rust
- **Excel処理**: umya-spreadsheet（Rustライブラリ）
- **書式保持の仕組み**: 「コピー」ではなく「不要シートを削除」方式を採用。元ファイルの書式情報をそのまま残す
- **対応OS**: Windows / Mac（Apple Silicon）
- **アプリサイズ**: 約6MB（Electronなら100MB超になるところ）

### 海外展開について

「SheetToolBox」という名称で**英語圏にも販売**する計画です。「ローカル完結でデータが外に出ない」という特徴は、GDPRなどデータ保護に敏感な海外市場でも強い訴求になると考えています。
